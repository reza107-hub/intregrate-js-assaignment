1.What is DOM and what is the purpose of DOM?

DOM means Document object model. It is used to manipulate Html. It gives us to access change to modify content and structure of webpage.
The purpose of DOM is provide a platform for programs to access and update the content and style of html. It enables interaction of the webpage.

2.How will you select HTML elements using DOM? Name the DOM methods.

We can select html elements using dom by using DOM methods. here are common method for selecting element:

  1. 1. getElementById() : we can select elements by a specific ID.
  2. 2. getElementsByClassName() : we can select elements by a specific class.
  3. 3. getElementsByTagName(): we can select elements by a specific html tag name.
  4. 4. querySelector() : Select the first element.
  5. 5. querySelectorAll(): Selects all elements. It returns a node list.

3.What is event bubble?

Event bubbling is the process by which an event is received by an element, and then subsequently bubbles up through the DOM tree to be received by the ancestors of that element.

1.What is a callback function and why will you use it?

A callback function is a function that is passed as an argument to another function and is executed after the parent function has completed its task. In JavaScript, callbacks are commonly used in event listeners. Using a callback function also allows for modular and reusable code.
